## 💬 协作功能

### @ 提及同事

工作项详情页 → 评论框输入 `@张三`：

1. 实时弹出成员联想下拉（点选直接插入）
2. 提交后：
   - 被 @ 的人收到**站内通知**（顶部铃铛红点 +1）
   - 被 @ 的人浏览器**右下角弹 toast**（WebSocket 实时推送）
   - 触发**飞书/钉钉/企微 webhook** 推送（如果配置了）

**示例**:
```
@张三 @李四 看一下这个任务，麻烦了！
```

> 提示: @ 自己不会重复通知；不存在的用户名静默忽略

### 工作移交

适用场景: 同事离职 / 转岗时把工作项批量转给接手人。

路径: `/handover`
- 选 fromUser → toUser
- 自动移交：未关闭工作项 + 评审参与 + 排期
- 接收人收到通知 + WebSocket 推送

### 飞书/钉钉机器人配置

1. 创建群机器人 (群设置 → 群机器人 → 添加 → 复制 webhook URL)
2. 系统设置 → **WebHook** (路径: `/webhooks`)
3. 新建 WebHook:
   - **Name**: 飞书 PM 群
   - **URL**: 粘贴 webhook URL
   - **Events**: 勾选 `comment.mention` / `workItem.*` / `risk.*`
4. 保存
5. 测试 → 点 "测试发送"
6. 触发 @ 提及 → 群里自动收消息卡片

> 飞书/钉钉/企微**自动识别** (URL 含 feishu.cn / dingtalk / qyapi.weixin.qq.com)

## 🛠️ 管理员功能

### 用户管理 (`/users`)

仅 admin (tenant_admin) 可访问：

- **新建用户**: 用户名(3-20位字母数字) + 显示名 + 初始密码 + 角色
- **改角色**: member / biz_admin / space_admin / tenant_admin
- **启停用户**: 停用后无法登录（**不能停用自己**）
- **重置密码**: 至少 6 位

### 审计日志 (`/audit-logs`)

仅 admin 可见：

- **时间** / **实体** / **操作** / **操作人** / **摘要**
- 筛选: 实体类型 (project/customer/...) · 操作 (create/update/delete) · 操作人 · 日期范围
- 切换 **表格/时间线** 视图
- 点 **详情** 看字段级 diff (老值 → 新值)
- 点 **跳转** 直接打开对应实体

**覆盖范围**:
- ✅ 项目的创建/更新/删除
- ✅ 客户/车型/联系人的变更
- ✅ 外部依赖的状态变更
- ✅ 用户登录/登出 (含失败)
- ✅ 角色变更

## 🤖 AI 助理用法

### 入口

右下角 🤖 按钮（任何页面都可用）

### 能力

#### 多轮对话
```
你: 领克 09 这个项目的风险怎么样？
AI: [自动调用工具查项目 + 风险预警] ...

你: 它的 PM 是谁？合同额多少？   (它 = 领克 09)
AI: [根据上文，PM 是 XXX，合同 600 万]
```

#### 智能填表
新建客户/车型/项目/工作项时，**AI 助理 → 智能填表**，描述一句话，自动填充字段。

#### 风险扫描
```
你: 扫一下所有项目的风险
AI: [调用 scan_risks] 找到 X 个高风险项目
```

#### 自然语言查询
```
你: 上周延期的工作项有哪些？
你: 张三本周排了多少小时？
你: 自动找一个 VIP 客户的所有需求
```

### 切换 LLM

右上角 → **AI 设置**：
- OpenAI / Anthropic Claude / DeepSeek / 通义千问 / 智谱 GLM / 月之暗面 / 字节豆包 / Ollama (本地) / 自定义
- 填 API Key + baseUrl + model
- "测试连接" 验证
- 选 "主 provider" → 立即生效

## 📅 甘特图用法

路径: `/gantt`

### 视图

- **缩放**: 日 (40px/天) / 周 (22px) / 月 (12px)
- **筛选**: 按项目 · 包含/排除未排期
- **显示未排期**: 工作项已创建但没排期的，会以虚线斜纹显示
- **显示依赖连线**: 红色=阻塞 / 蓝色=关联 / 灰色=重复

### 操作

- **拖时间条左/右边缘** = 改起止时间 (resize)
- **拖时间条中间** = 整体平移 (move)
- **双击时间条** = 弹精确编辑框 (日期选择器)
- **单击时间条** = 跳工作项详情
- **今天** = 红色竖线
- **周末** = 灰色背景

### 自动记录

每次拖动保存时：
- Activity 表记录排期变更
- 审计日志记录 PATCH 事件

## 🐛 常见问题

### Q: 浏览器刷新报错 "xxx is not defined"
A: 已知 Vite HMR bug，**已通过 SWC 插件修复**（V1.16）。如仍遇到，强刷一下 (`Ctrl+Shift+R`)。

### Q: AI 助理说"未配置"
A: 右上角 → AI 设置 → 选 provider → 填 key → 测试连接 → 选为主。

### Q: 怎么邀请同事？
A: admin 登录 → 用户管理 → 新建用户。同事用初始密码登录后改密码。

### Q: 飞书 webhook 推送没收到？
A: 检查 1) WebHook 是否 enabled 2) events 是否勾选 `comment.mention` 3) 群里机器人是否还在 4) 测试发送是否成功。

### Q: 数据怎么备份？
A: 见 DEPLOY.md 第 9.1 节，AVM 数据全在 `data/data.db` 单文件 SQLite，复制就行。

### Q: 能导出 Excel 吗？
A: 多个模块右上角有 "导出 Excel" 按钮（依赖/工作项/项目/风险）。
